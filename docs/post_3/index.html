Mesa plegable MDF

Este trabajo fue realizado en el ramo “Diseño y Mecanismos”. Tomó varias iteraciones llegar al modelo final, sin embargo la experiencia durante el proceso fue bastante buena y el resultado me dejó contento. Lo que destaco de este proyecto fue que aprendí lo básico de Inventor y lo útil de tener una plataforma de ensamblaje para testear prototipos.
