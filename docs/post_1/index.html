Casa Retro

La “Casa Retro” es un trabajo realizado durante el ramo de “Diseño de Espacios”. En base a los conocimientos de arquitectura entregados por el profesor Francisco Moure debíamos analizar un entorno, definir un usuario y diseñar un espacio para luego, exponerlo utilizando conceptos de planimetría y modelación 3D.
